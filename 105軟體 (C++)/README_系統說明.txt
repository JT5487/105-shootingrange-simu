# T91 雷射射擊系統 - 檔案整理說明

系統已經完成最終優化，並已將檔案重新分類整理，讓您的工作目錄保持整潔：

## 📂 核心執行檔案 (位於根目錄)
- **`啟動系統.bat`**：一鍵開啟「後端追蹤程式」與「管理控制台」。
- **`定位點測試.bat`**：一鍵開啟「後端追蹤程式」與「測試網格介面」。
- **`index.html`**：中央管理控制台 (管理射擊分數、重製、校準指令)。
- **`校正測試_v2.html`**：測試網格專用頁面 (顯示紅點、報位語音)。
- **`target_silhouette.png`**：靶板圖示資源。
- **`gunshot`**：Beretta 槍聲音訊檔案。

## 📂 系統目錄
- **`cpp/`**：包含所有 C++ 後端源碼、編譯腳本 (`Makefile.win`) 以及最重要的執行檔 `t91_tracker.exe`。
    - *校準檔 (`calibration_a/b.json`) 也會儲存在此處。*
- **`docs/`**：存放所有的系統說明文件、技術手冊與整合紀錄。
- **`archive/`**：存放舊版的 Python 腳本、過時的 HTML 頁面及備份檔案。
- **`logs_archive/`**：存放開發過程中的偵錯日誌檔與編譯產生的暫存檔。

---
### 💡 快速提示：
- 每次開啟系統前，請務必先執行 `啟動系統.bat`。
- 若更換電腦或移動目錄，請確認 `C:\msys64\mingw64\bin` 已加入系統環境變數，或參考腳本內部的路徑設定。

系統目前處於最佳狀態，祝您測試順利！

## 📂 語音提示系統 (Voice)
- **`voice/`**：存放所有的射擊流程語音指令檔 (如 `sop_01_ready.m4a` 等)。
    - **注意**：為確保瀏覽器相容性，所有音檔皆採用 **英文檔名** (如 `stop_01_cease.m4a`)。若需更換音檔，請保留原始檔名或同步修改 `index.html`。
- **SOP 運作機制**：
    1. 點擊「訓練開始」後，系統會依序播放 SOP 語音。
    2. 語音播放完畢後，才會自動發送 `start_scoring` 指令開始計分。
    3. 若中途點擊「停止射擊」，語音會立即中斷並播放停止口令。
