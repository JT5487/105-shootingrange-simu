### 功能 B4：自動校準

#### 前端整合（需實作）

**HTML 棋盤格顯示:**

<div id="chessboard-overlay" style="display:none; position:fixed; top:0; left:0; width:100vw; height:100vh; z-index:9999;">
    <canvas id="chessboard-canvas"></canvas>

**JavaScript 繪製棋盤格:**
```javascript
function drawChessboard(rows, cols) {
    const canvas = document.getElementById('chessboard-canvas');
    const ctx = canvas.getContext('2d');
    canvas.width = window.innerWidth;
    canvas.height = window.innerHeight;
    
    const squareWidth = canvas.width / (cols + 2);
    const squareHeight = canvas.height / (rows + 2);
    const startX = squareWidth;
    const startY = squareHeight;
    
    for (let row = 0; row < rows + 1; row++) {
        for (let col = 0; col < cols + 1; col++) {
            ctx.fillStyle = (row + col) % 2 === 0 ? 'black' : 'white';
            ctx.fillRect(
                startX + col * squareWidth,
                startY + row * squareHeight,
                squareWidth,
                squareHeight
            );
        }
    }
}
```

**自動校準流程:**
```javascript
async function autoCalibrate(camera) {
    // 1. 顯示棋盤格
    document.getElementById('chessboard-overlay').style.display = 'block';
    drawChessboard(6, 8);  // 6 行 8 列 = 5×7 內部角點
    
    // 2. 啟動後端自動校準
    await fetch('/command', {
        method: 'POST',
        body: JSON.stringify({
            action: camera === 'A' ? 'start_auto_calibration_a' : 'start_auto_calibration_b'
        })
    });
    
    // 3. 輪詢狀態（每秒）
    const interval = setInterval(async () => {
        const resp = await fetch('/command', {
            method: 'POST',
            body: JSON.stringify({ action: 'get_calibration_status' })
        });
        const data = await resp.json();
        
        const camKey = camera === 'A' ? 'camera_a' : 'camera_b';
        if (data.data[camKey].calibrated && data.data[camKey].mode === 'auto_chessboard') {
            clearInterval(interval);
            document.getElementById('chessboard-overlay').style.display = 'none';
            alert('自動校準完成！偵測到 ' + data.data[camKey].corners_detected + ' 個角點');
        }
    }, 1000);
    
    // 4. 10 秒逾時
    setTimeout(() => {
        clearInterval(interval);
        document.getElementById('chessboard-overlay').style.display = 'none';
        alert('校準逾時，請重試或檢查光線條件');
    }, 10000);
}
```

**UI 按鈕（加到 index.html）:**
```html
<button onclick="autoCalibrate('A')">🎯 自動校準相機 A</button>
<button onclick="autoCalibrate('B')">🎯 自動校準相機 B</button>
```

## 📌 常見問題 FAQ

### Q1: 如何修改命中容差？

**答:** 修改 `HitDetectionService.hpp:24-27`

```cpp
namespace HitDetectionConstants {
    constexpr double TOLERANCE_25M = 0.15;  // 改這裡
    constexpr double TOLERANCE_75M = 0.10;
    constexpr double TOLERANCE_175M = 0.20;
    constexpr double TOLERANCE_300M = 0.25;
}
```

重新編譯即可。

---

### Q2: 如何新增更多射手？

**答:** 需要修改多處（不建議，系統設計為 6 射手）

如果真的需要：
1. 修改 `HitDetectionConstants::NUM_SHOOTERS`
2. 修改 IRTracker 陣列大小（hits_[6] → hits_[N]）
3. 修改前端 UI
4. 重新測試所有功能

參考：`INVARIANTS.md` INV-DATA-001

---

### Q3: 自動校準失敗怎麼辦？

**可能原因:**
- 光線太暗或太亮
- 棋盤格尺寸不對
- 棋盤格被遮擋
- 相機失焦

**解決方案:**
1. 調整環境光線
2. 使用手動 4 點校準（原有方式）
3. 調整棋盤格尺寸（修改 ChessboardConfig）

參考：`IMPLEMENTATION_SUMMARY_A1_B4.md` 測試計劃

---

### Q4: 如何回退到舊版本？

**答:** 
1. 備份檔案在 git（若有）或手動備份
2. 移除新增的 .hpp 和 .cpp 檔案
3. 從 git 還原 IRTracker.hpp/cpp, main.cpp
4. 使用舊的 REBUILD.bat
5. 重新編譯

**建議:** 做任何大改動前先建立 git commit
