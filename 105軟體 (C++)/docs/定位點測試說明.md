# T91 雷射定位測試說明 (Standalone Testing Guide)

本文件說明如何在不使用 Unreal Engine 的情況下，完整啟動系統並測試雷射定位的準確性。

## 🚀 快速啟動
雙擊執行資料夾中的 **`定位點測試.bat`**。
執行後會自動開啟兩個網頁視窗：
1.  **射擊訓練控制台 (`index.html`)**：用來設定參數與啟動校正。
2.  **T91 系統校正與測試網格 (`calibration_grid.html`)**：用於接收雷射點與顯示紅點。

---

## 🛠️ 測試步驟

### 第一步：環境準備
1.  將 **`calibration_grid.html`** 視窗拖曳至您的**投影幕或主螢幕**。
2.  點擊網頁任意處，確保其進入 **「全螢幕模式」**。

### 第二步：雷射校正 (4-Point Calibration)
1.  切換到 **`index.html`** (控制台) 視窗。
2.  在右側「手動控制」區域，點擊 **「重新校準」 (Start Calibration)**。
3.  回到全螢幕的網格畫面，拿起模擬槍依序射擊四角的 **「紅色圓圈」**：
    *   順序：**🔴左上 → 🔴右上 → 🔴右下 → 🔴左下**
4.  射擊時，後端 Console 會顯示 `Camera A captured point X/4`。集滿四點即自動完成校正。

### 第三步：點位精準度驗證
1.  校正完成後，直接對著螢幕上的 **1~6 號黃圈** 進行射擊。
2.  **預期行為：**
    *   **螢幕反應**：在您射擊的位置會跳出一個 **「紅色光點」**，隨後自動消失。
    *   **分數反應**：控制台 (`index.html`) 對應的射手中彈數應該會自動增加。

---

## ❓ 常見問題
*   **雷射打中但沒出現紅點？**
    *   請檢查後端視窗是否有抓到點 (出現 `[DEBUG] ... IR points`)。
    *   若有抓到點但沒紅點，可能是「校正」沒做完或順序錯了，請點擊「重新校準」重做一次。
*   **紅點位置偏移很大？**
    *   請確保校正時，點位的順序完全正確（順時針方向：左上開始）。
    *   請確認 `calibration_grid.html` 是在全螢幕下進行校正。
