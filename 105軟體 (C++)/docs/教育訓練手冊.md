# T91 紅外線射擊訓練系統 - 教育訓練手冊

本手冊旨在協助教官快速熟悉「射擊訓練控制台」的操作流程與各項功能設定，確保訓練任務順利達成。

---

## 一、 系統啟動流程
1. **啟動後端程式**：進入本地目錄執行 `./t91_tracker` (或點擊 `啟動系統.bat`)。
2. **開啟控制頁面**：使用瀏覽器開啟 `index.html`。
3. **確認連線狀態**：檢查左側「系統連線」面板，確保「攝影機A/B」與「追蹤程式」顯示為綠燈。

---

## 二、 前端功能區塊說明

### 1. 訓練資訊輸入 (左上)
- **梯次/連隊/班別**：輸入當前受訓單位資訊，採單行水平佈局以節省空間。

### 2. 環境與模式設定 (左中)
- **環境設定**：提供一鍵切換「天候(晴天/雨天)」及「時段(日間/夜間)」。採圖文並茂設計，方便目視確認。
- **射擊模式**：提供 25m、75m、175m 及 300m 模式切換。按鈕採 2x2 網格放大佈局，提高操作便利性。

### 3. IR 偵測設定 (左下)
- **靈敏度 (閾值)**：水平式調整拉條。
  - **數值愈大**：愈嚴格，可過濾環境雜光干擾，但需配合雷射亮度。
  - **數值愈小**：愈靈敏，適合雷射較弱時使用。

### 4. 射手即時成績 (正中)
- **卡片佈局**：頂部為姓名輸入，中間左側為靶位圖像，中間右側為大型中彈數顯示（5em 特大尺寸），底部為狀態控制鍵。
- **靶位狀態**：
  - **正面(立靶)**：系統記錄擊中數。
  - **轉向(倒靶)**：靶位呈現 90 度轉向效果，此時不記錄中彈。
- **中彈回饋**：中彈時該射手卡片會閃爍，數字即時跳轉。

### 5. 戰術演練控制 (中下)
- **訓練開始**：啟動自動口令序列與計分核心。
- **停止射擊**：
    1. 立即停止計分。
    2. 執行語音任務：**「停止射擊！」** -> **「射手起立！」** -> **「向右轉，下射擊線！」**。
    3. 任務結束後，按鍵自動切換為藍色 **「上傳成績」** 與綠色 **「重新畫面」**。

### 6. 倒靶模式設定 (右上)
- **中彈即倒**：命中目標後靶板立即轉向。
- **秒倒靶**：輸入設定秒數，時間到全線自動轉向倒靶。
- **發倒靶**：射手打滿指定發數後，目標自動轉向倒靶。

### 7. 手動控制面板 (右下)
- **全部倒靶 / 全部立靶**：強制變更好全線狀態。
- **重置分數**：清除畫面成績（僅清除顯示，不影響已上傳之資料庫）。
- **重新校準 / 套用設定**：進行座標修正與後端參數更新。

---

## 三、 靶場操作程序指導 (SOP)
專為教官口令設計的輔助工具，位於螢幕最下方：
- **口令按鈕**：直接點擊即可播放對應軍事口令。
- **序列控制**：可手動切換至「下一動」或「上一動」預備位置。

---

## 四、 操作要領與注意事項
1. **全螢幕模式**：本介面已優化至全螢幕無捲軸，建議開啟瀏覽器 F11 全螢幕模式以獲得最佳視覺效果。
2. **成績上傳與下載**：訓練終止且口令播放完畢後，**務必點擊藍色的「上傳成績」**，系統將同步執行兩項動作：
    *   將數據寫入資料庫「射擊成績」表中。
    *   **自動儲存 xlsx 檔案**：下載標題為「射擊成績\_梯次\_連隊\_班別\_日期.xlsx」的 Excel 報表，內含所有射手詳細資訊與成績。
3. **系統重置**：若要進行下一波訓練，請點擊綠色的「重新畫面」或按 F5 重新載入，以重置所有射手狀態與計分。
4. **緊急處置**：若訓練中有安全疑慮，請立即按下紅色的「停止射擊」，系統將依序下達撤離指令並放倒所有靶位。
